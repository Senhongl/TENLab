	Environment
=========================
1. Download LibTorch from [1] and unzip
2. cmake > 3.11
3. llvm 11
4. OCaml 4.11


	Usage
=========================
1. revise test.tl
2. make
3. see LLVM IR in test.ll
4. see exec output in test.out

* clean: make clean

[1] https://download.pytorch.org/libtorch/cpu/libtorch-cxx11-abi-shared-with-deps-1.10.0%2Bcpu.zip
